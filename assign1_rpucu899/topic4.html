Topic 4 - Strings

A string is a sequence of characters (character is represented as an individual char). A string literal (or hard coded string) is represented by putting double quotes around the text. The type String is used for variables that will hold a string. In order to connect strings together, we can use a string concatenation operator (+). It can append strings to the end of each other and even a number to a string.

Commands we can use with strings are:

  • stringName.toUpperCase();
  • stringName.toLoweCase();
  • stringName.length();
  • stringName.charAt(index);
  • stringName.substring(inclusive index value, excluded index value);
