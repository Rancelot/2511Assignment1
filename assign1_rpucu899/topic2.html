Topic 2 - Accessing Libraries

In order to access these predefined libraries, we must import them into the program.

  • This must be done in every class that needs to access these libraries.
  • Importing is done through import declarations.

If you do not add an import statement you will get a syntax error.

  • The compiler will not know you are talking about a predefined class without an import statement.

Examples of Import Statements:

  • import java.util.Scanner
  • import java.io.*
