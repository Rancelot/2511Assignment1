Topic 5 - Loops

Loops will allow us to repeat a section of code a number of times. Like conditional statements, they are controlled by boolean expressions.

Java has three kinds of repetition statements: while, do, and for loops. The syntax of a while statement is as follows:

while (condition)
        statement;
while (condition)
{
         statement;
         statement;
         statement;
}
